1 - O que é um ataque DDoS e como posso me proteger?
DDos é fazer inumeras requisições a um servidor fazendo com que eventualmente ele fique indisponivel.
Para se proteger é possivel investir em largura de banda, ter uma conexão reserva, ter validações contra intruso.

5. Explique a diferença entre criar o repositório na nuvem e iniciar o repositório a partir de um código existente local.
Criar um repositorio na nuvem ele ficara salvo em um servidor diferente de sua maquina e o repositorio local ele existe na sua máquina.

6. Qual a diferença entre Git e GitHub?
Git é um gerenciador de repositorios e controle de versão de arquivos para códigos-fonte e desenvolvimento colaborativo.
Github é uma plataforma onde pode hospedar seus projetos e repositorios de forma online, onde é possivel vizualizar códigos de outros programadores.

7. Quais as principais diferenças entre o modelo ágil e o waterfall (modelo em
cascata)?
O modelo ágil tem foco na satisfação do cliente envolvendo ele durante o projeto.
O modelo Cascata é um processo que não precisa que os clientes participem.

11. O que é o protocolo HTTP? Qual a diferença entre HTTP e HTTPS?
  HTTP :O protocolo tem como função disponibilizar link de acesso para usuarios para que navegadores e servidores se comuniquem.
  Diferença entre um e o outro : O Http é focado na entrega de dados, se preocupando menos como a informação é transmitida
  tornando-o vunerável para ser invadido ou alterado, mas o HTTPS é mais seguro pois costuma ser oferecido pelo certificado
  SSL fornecendo uma conexão criptografada entre servidor e navegador garantindo mais segurança.
